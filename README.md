ROBLOX TO POLYTORIA CONVERTER
(made by m0scull3s on poly and roblox)


Ever found a cool clothing set on Roblox and you wanted to put it on your Poly avatar but couldn't because you were too lazy to redraw it?? Well fear no more! Now you can!

You can even convert Poly clothing to Roblox! Do note that it stretches it to make it fit.

Requirements:
Python 3 or higher
os module (py -m pip install os)
tkinter module (py -m pip install tkinter)
PIL module (py -m pip install PIL)

enjoy!!
